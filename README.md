# Tic Tac Toe (Python)

A beginner-friendly two-player Tic Tac Toe game built in Python using functions and input validation.

## Features
- Player vs Player gameplay
- Input validation for positions 1–9
- Win and draw detection
- Replay option after game ends

## File
- `tic_tac_toe.ipynb` – Jupyter notebook containing the full game

## Purpose
This project demonstrates basic Python programming concepts including:
- Functions
- Loops and conditionals
- Input validation
- Game logic and state tracking
